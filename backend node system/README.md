## Guideline of Employee Happiness Backend Database

###1)	Start Service
1.	use terminal
2.	cd Ehsql
3.	./Ehsql npm start

###2)	File description
1.	./Ehsql/db/DBConfig.js records the database information
2.	./Ehsql/db/usersql.js records the SQL script for each function
3.	./Ehsql/routers/users.js is the main file of node.js system, all the backend function are implemented here.
